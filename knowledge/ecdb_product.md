# ECDB 제품 정의

## 개요

ECDB(E-Commerce DataBase)는 글로벌 이커머스 데이터 SaaS 플랫폼이다.

## 제품 데이터 카테고리

### 1. 스토어 데이터

| 데이터 항목 | 설명 |
|-----------|------|
| 매출 | 온라인 스토어별 추정 매출 |
| 카테고리 비중 | 카테고리별 매출/트래픽 비율 |
| AOV | 평균 주문 금액 (Average Order Value) |
| Traffic | 방문자 수, 페이지뷰 |
| SEO Score | 검색엔진 최적화 점수 |

### 2. 기술 데이터

| 데이터 항목 | 설명 |
|-----------|------|
| Mobile App | 앱 유형, 다운로드 수 |
| Payment Provider | 결제 서비스 제공자 |
| Shipping Partnership | 배송 파트너사 |

### 3. 마켓 데이터

| 데이터 항목 | 설명 |
|-----------|------|
| 마켓플레이스 랭킹 | Revenue/GMV 기반 순위 |
| Market Overview | 국가/지역별 시장 개요 |
| Market Reports | 심층 시장 분석 리포트 |

### 4. Leads DB

| 데이터 항목 | 설명 |
|-----------|------|
| 담당자 연락처 | 온라인 스토어별 의사결정권자 정보 |
| 직함/역할 | eCommerce, D2C, Strategy 등 |

## 타깃 고객별 가치 제안

### 마켓플레이스 운영자

| 겪는 고통 | ECDB 솔루션 |
|----------|------------|
| 셀러 트렌드 파악 어려움 | 실시간 마켓플레이스 랭킹 |
| 경쟁 플랫폼 동향 모름 | 스토어 데이터 비교 |

### 리테일 전략기획

| 겪는 고통 | ECDB 솔루션 |
|----------|------------|
| 신시장 진출 의사결정 근거 부족 | 국가별 Market Report |
| 경쟁사 매출 추정 어려움 | 스토어 매출 데이터 |

### 물류 사업개발

| 겪는 고통 | ECDB 솔루션 |
|----------|------------|
| 이커머스 잠재 고객 발굴 어려움 | Leads DB |
| 시장 규모 산정 어려움 | 마켓 규모 데이터 |

### 브랜드 D2C

| 겪는 고통 | ECDB 솔루션 |
|----------|------------|
| 경쟁 브랜드 분석 어려움 | 스토어 데이터 |
| 채널별 성과 비교 불가 | 카테고리 분석 |

### 결제/핀테크

| 겪는 고통 | ECDB 솔루션 |
|----------|------------|
| 이커머스 성장 기회 포착 어려움 | 기술 데이터 |
| 파트너 발굴 어려움 | Leads DB |

### 정부/공공기관

| 겪는 고통 | ECDB 솔루션 |
|----------|------------|
| 산업 동향 파악 어려움 | Market Report |
| 정책 수립 근거 자료 부족 | 통계 데이터 |

- **데모 제안**: "15분 데모로 [specific_value]를 보여드릴 수 있습니다"
